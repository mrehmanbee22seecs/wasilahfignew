# Wasilah

This is a code bundle for Wasilah. Built with React, Vite, and Tailwind CSS.

## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.

## Building for production

Run `npm run build` to create a production build.

Run `npm run preview` to preview the production build locally.

## Deployment on Vercel

This project is configured for deployment on Vercel. To deploy:

1. Push your code to a GitHub repository
2. Connect the repository to Vercel
3. Vercel will automatically detect Vite and deploy the project

The project includes:
- `vercel.json` - Configured for SPA routing (all routes redirect to index.html)
- `vite.config.ts` - Build output configured for Vercel (dist folder)
